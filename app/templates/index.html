О приложении

Это простое клиент-серверное приложение для управления задачами, построенное на:

  • FastAPI - современный веб-фреймворк для Python
  • Jinja2 - шаблонизатор HTML
  • HTML/CSS - для веб-интерфейса

API эндпоинты

Приложение также предоставляет REST API:

  • GET /api/todos - получить все задачи в формате JSON
  • GET /api/todos/{id} - получить задачу по ID
  • POST /api/todos - создать новую задачу (JSON)
